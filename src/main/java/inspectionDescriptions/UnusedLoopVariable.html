Guidewire Software recommends to avoid unused loop variables in your code

A loop variable is unnecessary when using an interval
e.g.


for (i in 0..10) {
print("x")
}


can be replaced with (using the quick fix)


for (0..10) {
print("x")
}


When iterating over a list, but not using the loop variable... e.g.


for (i in list) {
print("x")
}


There is no quick fix for this
